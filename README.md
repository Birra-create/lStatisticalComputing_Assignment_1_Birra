## Project Description
This project is prepared as part of the **Statistical Computing** course.  
The main goal is to practice using **Git** and **GitHub** for managing and sharing R programming projects.  
It includes creating a Git repository, adding R Markdown files, committing changes, and pushing them to GitHub using **RStudio** and **PowerShell**.
